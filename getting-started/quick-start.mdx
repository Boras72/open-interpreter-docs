# Install

Run the following command in your terminal to install Open Interpreter.

> **Note:** You will need Python 3.10 installed on your system. Learn more.

```bash
pip install open-interpreter
```

# Run

### Terminal

After installation, you can easily start an interactive chat in your terminal by running:

```bash
interpreter
```

### Python

```python
import interpreter
interpreter.chat()
```
